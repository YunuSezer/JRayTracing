## How to Run?

1.  Install a Java IDE of your choice (Preferably IntelliJ).
2.  Clone the repo:
    ```bash
    git clone [https://github.com/yunusezer/JRayTracing.git]
    ```
3.   Run class `Main.java`.
4.  You can use mouse controls to interact with light source.
